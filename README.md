# Usage

grallect [options] host metric
    -c, --config FILE                Path to configuration file
    -v, --[no-]verbose               Run verbosely
    -h, --help                       Display this screen
        --version                    Show version

# Supported metrics

* CPU
* Disk
* DF
* Interface
* Memory
* Swap
* Load
